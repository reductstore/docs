# Deploy on ReductStore Cloud

<head>
  <link

ReductStore Cloud is available as a [SaaS offering](/solutions/cloud), allowing you to deploy ReductStore instances on-demand without managing the underlying infrastructure. This guide provides information about the architecture and instructions for deploying ReductStore as a SaaS instance on ReductStore Cloud.

## Next Steps

After deploying ReductStore as an instance, you can start storing and querying time-series data. Check our **[Guides](/docs/guides/index.mdx)** to learn more about the ReductStore features and how to use them.
